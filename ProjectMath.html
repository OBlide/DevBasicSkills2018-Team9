This is a web page to help you learn mathematics.

This is a number converter that converts numbers automatically to binary, decimal, octal and hexadecimal.


Just enter a number in its dedicated slot.

This is a combinatorics calculator that calculates the possibilities of given sets.


First select if you want to calculate permutations or combinations.

permutations stuff

Permutation without repetition : n!

Permutation with repetition : nPr (n, k), where n is the number of elements in the set and
k is the number of elements in the subset to be chosen.

combinations stuff
nCr (n, k), where n is the number of elements in the set and k is the number of elements in the subset to be chosen.
