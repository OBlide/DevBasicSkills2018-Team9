This is a web page to help you learn mathematics.

This is a number converter that converts numbers automatically to binary, decimal, octal and hexadecimal.


Just enter a number in its dedicated slot.

This is a combinatorics calculator that calculates the possibilities and combinations of given elements.


First select if you want to calculate permutations or combinations.

Permutation without replacement : n!, where n is the number of given elements.

Permutation with replacement : n^r, where n is the number of given elements in the set and
r is the number of elements in the subset to be chosen.

nCr (n, k), where n is the number of elements and k is the size of the subsets.
